# プロダクト要求定義書

## 誰のためのアプリか
締め切りが遠いと後回しにしてしまう、先延ばし癖がある自分自身のため。

## どんな課題を解決するか
- タスクの締め切りが遠いと危機感がなく、着手が遅れる
- タスクが大きいと「どこから始めればいい？」がわからず止まってしまう

## 解決策（差別化機能）
タスクを「30分でできる単位」に自動分解する機能。
大きなタスクを登録すると、今すぐ着手できる小さなステップに分割してくれる。

## 成功の定義
- タスクを登録したら30分単位のサブタスクに分解できる
- サブタスクを1つずつ完了にできる
- 全サブタスクが完了したら親タスクも完了になる
